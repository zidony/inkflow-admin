# InkFlow Admin 后台管理系统模板 (InkFlow Admin Template)

这是一套专为生产环境设计的现代、美观、轻量级的博客管理后台 UI 模板。本项目由资深前端架构驱动，摒弃了臃肿的第三方框架，致力于在极佳的视觉体验与极致的运行效率之间取得完美平衡。

## 🛠 技术栈 (Tech Stack)

* **核心框架:** [Bootstrap 5.3.8](https://getbootstrap.com/) (无 jQuery 依赖)
* **图标库:** [Bootstrap Icons 1.13.1](https://icons.getbootstrap.com/)
* **交互脚本:** 原生 JavaScript (Vanilla JS) - 零冗余依赖，极致性能
* **主题色调:** 沉稳蓝 / 清新绿 (Blue / Green Primary Colors)

## ✨ 核心特性 (Features)

* **现代化响应式布局:** * 经典的“侧边栏 + 顶部状态栏 + 主内容区”结构。
  * **丝滑收缩侧边栏:** 侧边菜单支持一键整体收缩与展开，自动适配不同尺寸屏幕。
* **高颜值顶栏:** 集成沉浸式状态栏，包含带头像的用户下拉菜单（支持展示个人资料、退出登录等操作）。
* **全局统一 CSS 架构:** 摒弃页面级的零散样式，全站采用一份深度定制、高度复用的核心 CSS 文件，免去后期去重与合并的烦恼，维护成本极低。
* **性能优先设计:** 界面精简克制，拒绝过度动画与无用模块，专为高频操作的生产环境优化。

## 📦 页面结构 (Pages Included)

本项目提供博客后台最核心的 3 个高频使用模板：

1. **`dashboard.html` (控制台首页)**
   * 数据概览：文章数、评论数、文件数等核心指标统计。
   * 快捷操作区：博客高频功能入口。
   * 动态流：最新发布文章列表、最新交互评论展示。
2. **`article-list.html` (内容管理 - 文章列表)**
   * 清晰的数据表格展现，支持状态展示与快速操作。
3. **`article-edit.html` (内容管理 - 文章发布/编辑)**
   * 高度整合的表单页面，发布新文章与编辑旧文章共用的精美 UI 模板。

## 🗂 菜单导航规划 (Navigation Map)

侧边栏已内置标准化的博客管理路由结构：
* **内容管理:** 文章列表、评论列表、栏目列表
* **媒体管理:** 相册列表、文件列表
* **用户管理:** 用户列表、权限设置
* **系统设置:** 站点信息、常规设置
